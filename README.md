# ЛапМаркет (LapMarket)

Маркетплейс и соцсеть для владельцев домашних животных.

## Идея продукта

| Направление | Описание |
|-------------|----------|
| Соцсеть | Лента постов владельцев, лайки, теги |
| Маркет | Товары для животных — **только сертифицированные продавцы** |
| Питомцы | Покупка породистых или усыновление |
| Услуги | Ветеринары, грумеры, дрессировка и др. |

### Обозначения животных

- **✓ С паспортом** — породистый, документы (РКФ / FIFe и т.п.)
- **♥ В добрые руки** — из приюта / бесплатно в любящую семью

## Стек

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- lucide-react

## Страницы MVP

| Путь | Раздел |
|------|--------|
| `/` | Главная, описание продукта |
| `/market` | Товары (фильтр: certified seller) |
| `/animals` | Питомцы с бейджами |
| `/services` | Ветеринары, грумеры |
| `/feed` | Соцлента |

## Дорожная карта

1. **Авторизация** — регистрация владельцев, продавцов, приютов
2. **Сертификация продавцов** — заявка, модерация, блок простых продавцов
3. **Каталог и корзина** — реальные товары, оплата
4. **Объявления о животных** — модерация, чат с продавцом
5. **Услуги** — календарь записи, отзывы
6. **Лента** — подписки, комментарии, медиа
