# ЛапМаркет (LapMarket)

Платформа для владельцев домашних животных: **цифровой паспорт питомца**, маркет, объявления, услуги и соцсеть.

## Главная фича — паспорт питомца

| Возможность | Описание |
|-------------|----------|
| Профиль питомца | Вид, порода, вес, чип, фото |
| Прививки и медкарта | История визитов и вакцинации |
| Напоминания | Cron + уведомления в приложении |
| Экспорт | PDF/печать паспорта, публичная read-only ссылка |
| Связь с услугами | Запись к ветеринару с привязкой к питомцу |

## Другие разделы

| Направление | Описание |
|-------------|----------|
| Соцсеть | Лента с привязкой постов к питомцам |
| Маркет | Товары от сертифицированных продавцов, корзина → запрос продавцу |
| Объявления | Породистые / в добрые руки, inbox для авторов |
| Услуги | Ветеринары, грумеры, отзывы после визита |

## Стек

- Next.js 16 (App Router), TypeScript, Tailwind CSS 4
- PostgreSQL + Prisma
- Auth.js (email / пароль, сброс пароля)
- Vercel Blob, Resend (email), PWA

## Запуск

1. Скопируйте [.env.example](.env.example) в `.env`
2. `docker compose up -d` или внешний PostgreSQL
3. Миграции и seed:

```bash
npm install
npm run db:migrate
npm run db:seed
npm run dev
```

## Основные маршруты

| Путь | Раздел |
|------|--------|
| `/pets` | Паспорта питомцев |
| `/pets/[id]` | Карточка: здоровье, напоминания, вес |
| `/feed` | Лента |
| `/market` | Каталог (+ корзина → запрос) |
| `/animals` | Объявления |
| `/services` | Услуги и запись |
| `/profile` | Профиль и дашборд |
| `/profile/inbox` | Входящие по объявлениям |
| `/onboarding` | Первый вход владельца |
| `/search` | Глобальный поиск |
| `/admin` | Модерация |

## Переменные окружения

- `DATABASE_URL`, `AUTH_SECRET`, `AUTH_URL`
- `BLOB_READ_WRITE_TOKEN` — загрузка фото
- `RESEND_API_KEY`, `EMAIL_FROM` — сброс пароля
- `CRON_SECRET` — `/api/cron/reminders` (Vercel Cron)
