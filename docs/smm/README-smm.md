# SMM-запуск LapMarket: инструкция

Материалы для бесплатного продвижения среди ветеринаров и грумеров в Telegram и ВКонтакте. Код сайта менять не нужно — только копируйте тексты из этой папки.

---

## Перед копированием

| Что | Значение |
|-----|----------|
| Сайт заявок **в тексте постов** | `lapmarket.vercel.app/for-business` |
| Ссылка **для кнопок** (ВК, меню, bio) | `https://lapmarket.vercel.app/for-business` |
| Telegram username | `@lapmarket_specialists` (запасной: `@lapmarket_vet`) |
| Короткий адрес ВК | `lapmarket_vet` → vk.com/lapmarket_vet |

**Комиссия (везде одинаково):** первые 10 специалистов в Москве — **0% на 2 месяца**; далее — **10% только с подтверждённой записи**.

**Язык:** простой русский, без слова «платформа» — лучше «сервис» или «сайт ЛапМаркет».

## Порядок действий (общий)

```mermaid
flowchart TD
  prep [Прочитать docs/smm]
  tg [Создать Telegram-канал]
  vk [Создать группу ВК]
  pin [Закрепить сообщения]
  week [Публиковать posts-week-1]
  prep --> tg --> pin
  prep --> vk --> pin
  pin --> week
```

1. Прочитайте константы выше и файл `telegram-structure.md`.
2. Создайте Telegram-канал и ВК-группу (можно в один день).
3. Опубликуйте и закрепите все сообщения из structure-файлов.
4. Следуйте графику из `posts-week-1.md` — 1–2 поста в день.

## Шаг 1. Telegram-канал

1. В приложении Telegram: **Создать канал** → тип «Канал».
2. Название: **ЛапМаркет | Для специалистов**.
3. Username: **@lapmarket_specialists** (если занят — `@lapmarket_vet`).
4. Вставьте **описание** из [telegram-structure.md](./telegram-structure.md).
5. Загрузите **аватар** (логотип с зелёным фоном).
6. Опубликуйте **4 сообщения** из того же файла — по одному.
7. На каждом сообщении: **Закрепить** (Pin).
8. Проверьте, что в описании канала есть ссылка `lapmarket.vercel.app/for-business`.

**Опционально позже:** через [@BotFather](https://t.me/BotFather) добавить кнопку «Оставить заявку» с URL `https://lapmarket.vercel.app/for-business`.

---

## Шаг 2. Группа ВКонтакте

1. На [vk.com](https://vk.com): **Создать сообщество** → **Группа**.
2. Название: **ЛапМаркет | Ветеринары и грумеры**.
3. Короткий адрес: **lapmarket_vet**.
4. Вставьте **описание** из [vk-structure.md](./vk-structure.md).
5. Аватар и обложка — по рекомендациям в файле.
6. Опубликуйте **3 поста** и **закрепите** каждый.
7. **Меню сообщества:** пункт «Оставить заявку» → `https://lapmarket.vercel.app/for-business`.

---

## Шаг 3. Первая неделя контента

Откройте [posts-week-1.md](./posts-week-1.md).

| День | Действие |
|------|----------|
| 1 | Пост «пустые окна груминг» → TG + ВК |
| 2 | Пост «звонки ветеринария» → TG + ВК |
| 3 | Кейс «Здоровый хвост» → TG + ВК |
| 4 | Опрос → в ВК создать опрос, в TG — просить ответ в комментариях |
| 5 | Неявки → TG + ВК |
| 6 | Кто мы → TG + ВК |
| 7 | Итог недели → TG + ВК |

Один и тот же текст можно публиковать в обоих каналах в один день (утро TG, вечер ВК) или на следующий день во втором канале.

## Метрики без рекламного бюджета

Что можно отслеживать бесплатно:

- **Просмотры** постов в Telegram (статистика канала) и ВК (охват записей).
- **Подписчики** — рост за неделю.
- **Комментарии и опрос** (день 4) — вовлечённость.
- **Заявки на сайте** — сколько форм пришло с `/for-business` (если настроены `PARTNER_NOTIFY_EMAIL` и Resend).

**UTM (по желанию):** в кнопках ВК используйте  
`https://lapmarket.vercel.app/for-business?utm_source=vk&utm_medium=social`  
и для Telegram `utm_source=telegram` — так в аналитике сайта видно источник.

## Чего не обещать в постах

| Нельзя писать как «уже есть» | Как писать честно |
|------------------------------|-------------------|
| Автонапоминания клиентам | «Скоро добавим напоминания» |
| Онлайн-оплата через ЛапМаркет | «Оплата в клинике напрямую» |
| Гарантия N клиентов в месяц | «Зависит от города и профиля» |
| Работа во всех городах России | «Сейчас акция 0% — для первых 10 в **Москве**» |

## Связь с сайтом

- Страница партнёров: `/for-business` (форма → API `partner-request`, письмо на `PARTNER_NOTIFY_EMAIL`).
- Каталог для клиентов: `/services`.
- Регистрация специалиста: `/register` (роль специалист).

Тексты в SMM согласованы с блоками на `/for-business`, но формулировки про комиссию в соцсетях **полнее** (0% → 10%) — при обновлении сайта сверяйте оба места.
